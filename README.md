解压```如果提示没有权限，加上`sudo`以`root`用户的身份再运行。执行完就可以在`/usr/local/`下看到`go`目录了。**配置环境变量：** Linux下有两个文件可以配置环境变量，其中/etc/profile是对所有用户生效的；$HOME/.profile是对当前用户生效的，根据自己的情况自行选择一个文件打开，添加如下两行代码，保存退出。```shellexport GOROOT=/usr/local/goexport PATH=$PATH:$GOROOT/bin```修改`/etc/profile`后要重启生效，修改`$HOME/.profile`后使用`source`命令加载`$HOME/.profile`文件即可生效。 检查：![img_2.png](img_2.png)### <a name="4">4.GOROOT和GOPATH</a>
